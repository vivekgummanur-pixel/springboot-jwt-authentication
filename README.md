# Spring Boot JWT Authentication

## Technologies
- Java
- Spring Boot
- Spring Security
- JWT
- H2 Database
- Maven

## Features
- User Registration
- User Login
- JWT Token Generation
- JWT Validation
- Protected APIs
- Spring Security Integration

## How to Run

1. Clone the repository
2. Open in Eclipse/IntelliJ
3. Run the application
4. Test APIs using Postman
